# 📢 Amit Alerts System - Home Front Command for FiveM

מערכת התראות בזמן אמת המבוססת על נתוני פיקוד העורף, המותאמת לשרתי FiveM. הסקריפט מאפשר לשחקנים להגדיר אזורי התרעה מותאמים אישית ולקבל עדכונים חיים בתוך השרת.

## ✨ פיצ'רים עיקריים
- **ניהול אישי:** פקודת `/alerts` הפותחת תפריט מתקדם להגדרת אזורי התרעה.
- **סנכרון מלא:** קבלת התראות אמת מבוססות API של פיקוד העורף.
- **ממשק משתמש (UI):** תפריט בחירה נוח וידידותי למשתמש (RTL Support).
- **אופטימיזציה:** צריכת משאבים מינימלית (Resmon Low).

## 🚀 התקנה

1. הורד את קבצי הסקריפט והעבר אותם לתיקיית ה-`resources` בשרת שלך.
